## ❗️ Troubleshooting

| Problem               | Solution                                       |
|----------------------|-----------------------------------------------|
| Game won't start     | Run as Administrator / Update DirectX          |
| Performance drops    | Disable "Screen Shake" or lower VFX quality   |
| Controller not found | Connect controller before launching the exe    |
| Antivirus warning    | Add the game folder to exclusions             |
| Save sync error      | Run the save_manager tool in the /tools folder |
